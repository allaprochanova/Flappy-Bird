🐦 Flappy Bird – Консольная версия на 7 языках
Легендарная игра-флаппер прямо в вашем терминале!
Управляйте птичкой, пролетайте между трубами и устанавливайте рекорды.
Реализована на семи языках программирования с единым геймплеем, цветной графикой, счётчиком очков, таблицей рекордов, настраиваемой сложностью и звуковыми эффектами.

🚀 Возможности
Игровой процесс – птица летит вправо, трубы появляются слева, игрок нажимает пробел, чтобы подняться.

Физика – гравитация, подъём при нажатии, столкновение с трубами и границами экрана.

Счёт – увеличивается при успешном пролёте между трубами.

Рекорд – лучший результат сохраняется в локальный файл.

Сложность – скорость игры увеличивается с каждым пройденным препятствием.

Цветной вывод – птица, трубы и фон окрашены для лучшей визуализации.

Управление – пробел / клавиша вверх для прыжка, Q для выхода, R для перезапуска.

Звуковые сигналы – при столкновении и прыжке (в некоторых реализациях).

Кроссплатформенность – работает в любом терминале (Linux, macOS, Windows с поддержкой ANSI).

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [опции]
Опции
Опция	Описание
-s, --speed <N>	Базовая скорость игры (задержка в мс, по умолчанию 100)
-h, --help	Справка
Управление
Клавиша	Действие
Пробел / ↑	Прыжок (птица поднимается вверх)
Q	Выход из игры
R	Перезапуск после окончания игры
🛠 Установка и запуск
Python
bash
python flappy.py [опции]
Требуется Python 3.6+ (использует стандартную библиотеку curses на Unix; на Windows использует windows-curses).

C++
bash
g++ -std=c++17 flappy.cpp -o flappy -lncurses
./flappy [опции]
Требуется библиотека ncurses (установите через пакетный менеджер).

Go
bash
go build flappy.go
./flappy [опции]
Использует github.com/nsf/termbox-go (установите: go get github.com/nsf/termbox-go).

JavaScript (Node.js)
bash
npm install blessed
node flappy.js [опции]
Использует библиотеку blessed для работы с терминалом.

C#
bash
csc flappy.cs
mono flappy.exe [опции]   # или dotnet run
Использует стандартный Console с ANSI-кодами (требуется поддержка).

Ruby
bash
ruby flappy.rb [опции]
Использует встроенную библиотеку curses.

Java
bash
javac flappy.java
java flappy [опции]
Использует lanterna (установите через Maven или скачайте JAR).

🧠 Логика игры
Инициализация – птица находится в центре экрана, трубы генерируются с интервалом.

Гравитация – птица постоянно падает вниз с ускорением.

Управление – при нажатии пробела птица получает импульс вверх.

Трубы – появляются справа и двигаются влево; имеют разную высоту и разрыв.

Столкновения – касание трубы, верхней/нижней границы экрана приводит к завершению игры.

Счёт – увеличивается, когда птица пролетает мимо трубы (середина разрыва).

Рекорд – сохраняется в файл и отображается в игре.

Сложность – скорость движения труб и гравитация постепенно увеличиваются.

✨ Дополнительные фичи
Адаптивная сложность – игра становится быстрее с каждым очком.

Цветовая схема – птица жёлтая, трубы зелёные, фон тёмный.

Звук – при прыжке и столкновении воспроизводится системный бип.

Пауза – нажмите P, чтобы приостановить игру.

Меню – отображается рекорд и инструкция перед началом.

Автосохранение – рекорд обновляется автоматически.
